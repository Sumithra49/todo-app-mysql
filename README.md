##  Features

- Create a to-do item
- View all to-do items
- Toggle complete/incomplete
- Delete a to-do item
- Connected to MySQL using Sequelize

 # .env
-  DB_HOST=your_db_host
- DB_PORT=3306
- DB_USERNAME=your_db_user
- DB_PASSWORD=your_db_password
- DB_DATABASE=your_db_name
- PORT=
